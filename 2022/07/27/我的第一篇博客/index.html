hexo+github搭建个人博客

发表于 2022-07-27 阅读次数：

前端工程师通过hexo+github部署自己的个人博客流程

github新建项目

  • xxljunjun.github.io

全局安装hexo-cli

新建本地文件夹blog

  • hexo init：初始化项目
  • hexo g：生成网站静态文件
  • hexo s：启动项目

将Hexo部署到GitHub

  • 更改_config.yml
  • npm install hexo-deployer-git –save

可以通过github访问

更换主题

  • 在blog/themes文件夹中
  • 配置_config.yml(称为站点配置文件)

写文章

  • hexo new “我的第一篇博客”

主要参考文章

https://zhuanlan.zhihu.com/p/102592286
