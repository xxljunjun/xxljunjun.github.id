javascript数组常用方法

发表于 2022-09-14 更新于 2022-10-07 分类于 web前端 阅读次数：

改变原数组：push、pop、shift、unshift、splice、reverse、sort
其他：join、indexOf、concat、toString、slice、includes、reduce、filter、map、forEach、isArray

不改变原数组的方法

  • array.join()
  • array.indexOf(item)
  • array.concat(数组)
  • array.toString()
  • array.slice(start,end)
  • array.includes(item)
  • array.reduce(function)
  • arr.filter(function)
  • arr.map(function)
  • arr.forEach(function)
  • Array.isArray

改变原数组的方法

  • array.push(item,item2…)
  • array.pop()
  • array.shift()
  • array.unshift(item,item2…)
  • array.splice(索引，删几个，添加1个，添加2个)
  • array.reverse()
  • array.sort()
